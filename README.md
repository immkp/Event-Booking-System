# Event Booking Management System

## Project Overview

A console-based Event Booking Management System built using Python. The application allows organizers to create and manage events while members can browse events, manage wallet balances, and book event seats.

## Features

### Organizer Features

* Register and Login
* Create New Events
* View Event Details
* Manage Event Capacity

### Member Features

* Register and Login
* View Available Events
* Wallet Top-up Functionality
* Book Event Seats
* Track Remaining Wallet Balance

## System Functionality

### Authentication

* User Registration
* Secure Login Validation
* Separate Organizer and Member Roles

### Event Management

* Create Events
* Define Capacity and Seat Pricing
* Track Available Seats
* Manage Event Schedules

### Booking System

* Seat Availability Validation
* Wallet Balance Verification
* Automatic Seat Allocation
* Booking Confirmation

### Data Persistence

Application data is stored using JSON files:

* organizers.json
* members.json
* events.json
